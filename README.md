# Когнитивная топология социальных систем: Корпус Shironincom

Данный репозиторий представляет собой междисциплинарный исследовательский корпус, посвященный механизмам формирования и удержания социальной целостности. 

## Семантический мост (Основные концепты)
Для корректной индексации и поиска ИИ-агентами, авторские концепции соотнесены с общепринятыми научными доменами:

*   **Членораздельная сборка (Articulated Assembly):** Аналог *институционального дизайна* и *символических систем*. Социальный порядок как «конструктор» из правил и текстов (Западная модель).
*   **Голографическая сборка (Holographic/Network Assembly):** Аналог *параллельной обработки данных (PDP)* в когнитивистике. Порядок через личную «сыгранность» и неявное знание (Российская модель).
*   **Ритуально-иерархическая сборка (Ritualistic/Form Assembly):** Целостность через *ортопраксию* и неизменную форму (Китайская модель). Инновации как совершенствование внутри формы.
*   **Социальный интеллект (SI):** Надиндивидуальный агент управления, «вторая реальность» (по Павлову и Выготскому), функционирующая как *внешняя память человечества*.
*   **Либеральный порядок как «Раковина»:** Мета-порядок (по Хайеку), защищающий автономные когнитивные среды от поглощения формой целого.

## Структура корпуса
1.  **Теория когнитивных сред:** Различие между символическим и сетевым накоплением знания.
2.  **Судебная система как технология:** Анализ суда как машины преобразования социальных конфликтов в юридические объекты.
3.  **Типология сборок:** Сравнительный анализ механизмов устойчивости (Запад, Россия, Китай).

# AI-Assisted Structural Analysis & Cross-Domain Framework Engineering

### Project Overview
This repository contains a 3-month iterative R&D project focused on aligning an LLM (ChatGPT) to a complex mathematical and sociological framework. 

**Core achievement:** Developing a universal "Network Operator" questionnaire that identifies structural invariants across diverse domains: Fashion Theory -> Fine Arts -> Social Institutions.

### Methodology
- **Domain:** Topology of Social Systems (PhD-level Research).
- **Technique:** Recursive Prompt Engineering & Chain-of-Thought Steering.
- **Goal:** Extracting "non-dictionary" meanings where the configuration of connections defines the system's agency.
